# Security Policy

## Reporting a Vulnerability

Please do not open a public issue for security-sensitive reports.

Email security reports to:

```text
security@paintnode.com
```

Include:

- affected PaintNode version or commit
- operating system and architecture
- reproduction steps
- impact summary
- any relevant files, logs, or screenshots

We will acknowledge valid reports as soon as practical and coordinate fixes before public disclosure.

## Supported Versions

PaintNode is in early MVP development. For now, security fixes target the latest published release and the current `main` branch.
