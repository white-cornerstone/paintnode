# Code of Conduct

PaintNode is an early open-source project. We want discussion, issues, and pull
requests to stay useful, respectful, and focused on improving the editor.

## Expected Behavior

- Be respectful of different experience levels, backgrounds, and viewpoints.
- Keep feedback specific, technical, and actionable.
- Assume good intent, but accept correction when your words or actions cause harm.
- Stay on topic in issues, pull requests, reviews, and discussions.
- Report security-sensitive problems privately through `SECURITY.md`.

## Unacceptable Behavior

- Harassment, threats, hate speech, or personal attacks.
- Repeatedly derailing project discussions.
- Publishing private information without permission.
- Spam, scams, or intentionally disruptive behavior.
- Attempts to sneak malicious code, secrets, or unrelated payloads into the project.

## Enforcement

Project maintainers may edit, hide, or delete comments; close issues or pull
requests; block users; or report abuse to GitHub when needed to keep the project
safe and productive.

If you see a problem, contact the maintainers through the repository issue
tracker for public conduct concerns, or use the private security contact in
`SECURITY.md` for vulnerability reports.
