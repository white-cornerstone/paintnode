What's new in PaintNode 0.2.0

- Rebuilt the AI workflow around provider SDKs, with PaintNode-owned image-generation tools instead of direct local CLI calls for Codex and Claude.
- Added PaintNode-managed Codex and Claude runtimes. PaintNode can download, update, and launch the supported provider runtime without requiring Terminal setup; existing local installations remain available as an advanced option.
- Separated the AI Director from image generation so you can choose the planning provider independently from the image provider, with reusable profiles and structured Director actions.
- Added persistent AI Director sessions, review previews, capability discovery, and clearer provider progress throughout AI workflows.
- Improved workflow artifacts, layer/asset decoupling, generative fill orchestration, retouching, upscaling, and AI settings navigation.
- Kept Antigravity available as an external installation while it continues to use its existing connection path.
