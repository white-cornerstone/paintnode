What's new in PaintNode 0.1.2

- Added a first-run AI setup wizard to help users choose and test their local AI provider.
- Improved AI provider controls, including clearer Codex/Antigravity setup paths and removal of the retired minimal reasoning option.
- Fixed macOS Finder document opening so double-clicking an `.ora` file opens it in PaintNode after launch.
- Improved macOS Quick Look behavior for OpenRaster files, including larger previews while keeping lightweight Finder thumbnails.
- Added PaintNode branding and GitHub release presentation assets.
