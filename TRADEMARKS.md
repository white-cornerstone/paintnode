## Allowed Uses

You may use the PaintNode name to:

- Refer truthfully to the PaintNode project.
- Link to the official repository, website, documentation, or releases.
- Describe compatibility with PaintNode.
- State that your work is a fork of PaintNode, as long as it is clear that it is not the official PaintNode app.

## Modified Versions And Forks

If you distribute a modified version or fork, you must make it clear that it is not the official PaintNode release.

Do not use the PaintNode name, logo, icon, signing identity, updater channel, website, or official release branding for a modified build without written permission from White Cornerstone Pty Ltd.

Modified builds should use their own application name, icon, bundle identifier, signing identity, update channel, website, and support contact.
